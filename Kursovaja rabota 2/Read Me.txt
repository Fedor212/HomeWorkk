Программа высчитывает площадь фигуры ограниченно тремя функциями:
y = 0.6x+3
y = (x-2)(x-2)(x-2) - 1
y = 3/x

Так же высчитывает необходимые для решения задачи корни пересечения 
этих функций между собой и с осью абсцисс (всего 5 корней).

Поддерживает следующие запросы командной строки:
./report -e1 0.001 -e2 0.002 - для вычисления искомой площади с заданными точностями
(-е1 0.001 точность вычисления корней,  -e2 0.002 - точность вычисления площади).
./report -h  - для отображения Help информации. 
./report -r 0.001  - для вычисления всех 5-ти корней с заданной точностью 0.001.
./report -r для тестирования функции поиска корней двух тестовых функций: y = x*x*x+2 и y = -х+3
Ожидаемый результат примерно 0.6823.
./report -i для тестирования функции вычисления площади тестовой функции на отрезке [0;3]: y = -х+3
Ожидаемый результат примерно 4.5.
